# Delivery Checklist

## 需求与范围

- [ ] 已判断快速模式 / 完整模式
- [ ] TAPD 目标已确认
- [ ] 主单/子单关系已确认
- [ ] 验收标准已提取
- [ ] 不涉及范围已说明
- [ ] 一页式任务启动卡已完成

## 代码与影响

- [ ] 涉及仓库已确认
- [ ] worktree 路径已确认
- [ ] CodeGraph 索引视角已确认
- [ ] 公共符号已做 impact
- [ ] 跨模块影响已说明

## 验证

- [ ] 已说明证据等级
- [ ] 已列出已执行验证
- [ ] 已列出未执行验证
- [ ] 已说明残余风险

## 交付

- [ ] 已按交付物矩阵判断必须产出的文档
- [ ] H5 / 前端联调说明已准备或已明确不涉及
- [ ] 测试说明已准备
- [ ] 产品说明已准备
- [ ] 部署说明已准备
- [ ] TAPD 回写策略已确认

## 复盘

- [ ] 任务复盘已完成或已明确暂不需要
